**基本指令**

```
 hexo clean  // 清除public文件

hexo generate // 生成public文件

hexo deploy // 远程部署

hexo server // 本地部署

hexo douban -bgm //爬取豆瓣账号上的书籍/游戏/电影

hexo new "文章名" //默认生成新post

hexo new draft //生成草稿

hexo new page //生成x

StarKang1.github.io

```
