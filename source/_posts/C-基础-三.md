### 二维数组

就是再一维数组的基础上增加一个维度

#### 定义方式

```
1.数据类型 数组名[行数] [列数]
2.数据类型 数组名[行数] [列数] = {{数据1，数据2}，{数据3，数据4}}
3.数据类型 数组名[行数] 列数]={数据1，数据2，数据3，数据4}
4.数据类型 数组名[] [列数]= {数据1，数据2，数据3，数据4}

```
#### 二维数组作用

1.查看占用内存空间
2.查看二维数组的首地址

二维数组应用案例

考试成绩统计：


```
int score[3][3] = {
    {100,100,100},
    {90,50,100},
    {60,70,80}
};
string name[3] = {"张三","李四","王五"};
for (size_t i = 0; i < 3; i++)
{
    /* code */
    int num = 0;
    for (size_t j = 0; j < 3; j++)
    {
    /* code */
        num+=score[i][j];
    }
    cout << "第" << i+1 << "个人" << name[i] << "的考试总成绩为" << num << endl;
}
```
